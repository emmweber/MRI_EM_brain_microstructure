# MRI_EM_brain_microstructure

This repository contains the notebook, pretrained models and data to replicate the results in the manuscript “Non-Parametric Prediction of Brain MRI Microstructure using Transfer Learning” by G. Chau* , E. Weber*, A. Batra, L. Ni, M. Zeineh, A. Chaudhari, E. Adeli, J. Knowles and J. McNab, published in Imaging
Neuroscience, 3 (2025).
